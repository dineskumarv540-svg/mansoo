# Contributing to Mansoo

Thank you for your interest in contributing to **Mansoo – The Voice of Heart**!

## Development Setup
1. Clone repository: `git clone https://github.com/dineskumarv540-svg/mansoo.git`
2. Navigate to application folder: `cd mansoo-expo`
3. Install dependencies: `npm install`
4. Start development server: `npm start`

## Pull Request Process
- Create feature branch: `git checkout -b feature/my-new-feature`
- Ensure linter passes: `npm run lint`
- Submit PR referencing target issue.
