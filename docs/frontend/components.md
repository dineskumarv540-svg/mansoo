# Atomic UI Component Library Guide

Residing under `src/components/ui/`:

- **`AppButton`**: Variants (`primary`, `secondary`, `outline`, `ghost`, `pro`), sizes (`sm`, `md`, `lg`), loading spinner, icon support.
- **`AppInput`**: Label, validation error label, left/right icons, multiline.
- **`AppAvatar`**: Circular avatar with preset sizes, story gradient border, verified/pro badge overlays.
- **`AppBadge`**: Badges (`pro`, `verified`, `error`, `success`, `secondary`), numeric limiters.
- **`AppChip`**: Selectable category/hashtag filter chips with active gradient fill.
- **`AppToast`**: Animated snackbar notifications (`success`, `error`, `warning`, `info`).
- **`AppDialog`**: Confirmation modal with backdrop darkening.
- **`AppEmptyState`**: Empty feed & search results placeholder.
- **`AppFAB`**: Floating Action Button with linear gradient fill.
- **`SettingsTile`**: Configurable row tile for settings list items and switches.
