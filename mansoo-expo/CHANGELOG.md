# 📝 Mansoo CHANGELOG

All notable changes to the **Mansoo – The Voice of Heart** React Native application will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2026-07-24

### 🚀 Added
- **Design System & Theme Tokens**: Strict 4px grid spacing, standardized typography scale, HSL dark mode palettes, and dynamic `ThemeContext`.
- **Atomic UI Components**: Reusable `AppButton`, `AppInput`, `AppAvatar`, `AppBadge`, `AppChip`, `AppToast`, `AppDialog`, `AppEmptyState`, `AppFAB`, and `SettingsTile`.
- **Feature 1 — Writing Rooms**: Live collaborative writing rooms, prompt challenges, live participant chat, typing indicators, moderator controls, and direct feed publishing.
- **Enterprise Security Suite**: Hardened Firestore & Storage security rules, callable Cloud Functions validation (`zod`), profanity moderation (`bad-words`), rate limiting, and encrypted local storage via `expo-secure-store`.
- **Legal & Compliance**: Added Privacy Policy, Terms & Conditions, Community Guidelines, and Play Store compliant account deletion flow.
- **Customer Support Ecosystem**: In-app Help Center, FAQ accordions, ticket tracking system, and `SupportTicketModal`.
- **Documentation Hub**: `firebase_architecture.md`, `developer_guide.md`, `master_audit_report.md`, `enterprise_architecture_hub.md`, and `openapi.json`.

## [0.9.0] - 2026-07-23

### 🚀 Added
- Analytics+ Dashboard with weekly visitor charts and reading time breakdown.
- Gamification Engine (Writing Streaks, XP points, writer levels, achievement badges, leaderboards).
- Mansoo Premium with Gold PRO badges, ad-free experience, and Google Play Billing integration.
- Professional Social Sharing (HD story exporter, social deep links, post/profile QR codes).
- Multilingual support for English, Hindi, Hinglish, Urdu, and Russian with RTL layout support for Urdu.
